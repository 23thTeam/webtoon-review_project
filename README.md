# 웹툰어스 (Webtoon us)
- 📖 웹툰 리뷰 웹사이트
- 웹툰의 리뷰를 작성하고, 유저 별 작성 리뷰와 웹툰 별 리뷰 및 정보를 확인할 수 있는 웹사이트입니다.

# 주요 기술
- HTML, Jinja2, JavaScript
- Flask
      
# 팀원
- 김노을
- 배근우
- 정은화
